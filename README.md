##  Project Overview
This project is a personal financial dashboard created in Power BI. It is designed to simulate, forecast, and visualize potential future earnings and career progression milestones. 

** Data Disclaimer:** The dataset used in this project is entirely simulated (dummy data). I generated this hypothetical data specifically for the purpose of practicing Power BI visualization techniques, data modeling, and DAX functions. It does not reflect any real financial records.

##  Tools & Technologies
- **Data Visualization & BI:** Power BI Desktop
- **Calculations:** DAX (Data Analysis Expressions)
- **Data Generation:** Custom simulated dataset

##  Key Features & Analysis
- **Salary Forecasting:** Visualizing potential income growth and scenarios over time.
- **Trend Analysis:** Tracking cumulative earnings and hypothetical financial goals.
- **Interactive Dashboards:** Utilizing slicers and filters to dynamically adjust the simulation parameters.
